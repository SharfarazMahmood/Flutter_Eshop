# E-Shop

This is a Flutter Demo project.

## Features
- Signup, Login
- App menu
- Shop page
  - All products view
  - Favorite products view
- Products details page
- Add to cart, add to favorites options
- Cart page 
  - Total cost, order option, slide left to delete a cart item
- Orders page
- Manage products
  - List of products added to the shop by individual user, 
  - Add new products and Edit existing products
